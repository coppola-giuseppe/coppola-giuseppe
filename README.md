## 🚀 Progetti in Evidenza

### 🐉 [D&D Manager](https://github.com/coppola-giuseppe/Progetto-Programmazione-Avanzata)
*Applicazione Client-Server distribuita per la gestione di campagne D&D 5e e contenuti Homebrew.*
* **Stack:** Java, Spring Boot, JavaFX, MySQL.
* **Highlight:** Frontend desktop asincrono sviluppato per non bloccare la UI, Backend RESTful gestito con Spring Boot , manipolazione di payload JSON e Base64 per il trasferimento di immagini, integrazione nativa con la *D&D 5e SRD API* e suite di unit test automatizzati con JUnit.

### 📋 [Kanban Distributed App](https://github.com/coppola-giuseppe/Progetto-Reti-Informatiche)
*Applicazione distribuita concorrente per la simulazione e gestione di task tramite metodo Kanban.*
* **Stack:** C, Socket API, Linux (Ubuntu).
* **Highlight:** Architettura di rete ibrida (Client-Server per la sincronizzazione centrale e Peer-to-Peer per le fasi di review tra nodi) , gestione autonoma della concorrenza, sistemi di tolleranza ai guasti tramite heartbeat (PING/PONG) e protocollo di comunicazione custom su socket.

### 🎓 [Laureandosi 2.0](https://github.com/coppola-giuseppe/Progetto-Ingegneria-del-Software)
*Sistema gestionale OO per l'automazione, il calcolo delle carriere e l'emissione dei prospetti di laurea.*
* **Stack:** PHP, JSON, FPDF, PHPMailer.
* **Highlight:** Progettazione accurata del software basata su analisi dei requisiti e diagrammi UML. Elaborazione algoritmica di carriere universitarie con filtri dinamici , generazione procedurale di PDF , motore di invio mail temporizzato anti-spam e suite integrata di test di regressione backend.

### 🎮 [Piattaforma Retro Gaming (Progettazione Web)](https://github.com/coppola-giuseppe/Progetto-Progettazione-Web)
*Piattaforma web full-stack per sfidare il computer in classici giochi.*
* **Stack:** PHP, MySQL, JavaScript, HTML5/CSS3.
* **Highlight:** Include i giochi *Tris (Tic Tac Toe)*, *Impiccato* e *Sasso Carta Forbici*. Implementa un sistema completo di autenticazione (registrazione, login, recupero credenziali) focalizzato sulla sicurezza informatica, con hashing e cifratura delle password eseguita sia lato client (front-end) sia lato server (back-end), oltre al tracciamento storico delle partite.
